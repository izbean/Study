# 제네릭

--- 
### 주로 사용하는 약어 의미

E: Element | T: Type | V: Value | K: Key

--- 
### 언바운드 와일드카드 타입 특성
 - List<?> 와 같이 ?로 표현

--- 
### 바운드 와일드카드 타입 특성
 - 무공변: 오로지 자기 타입만 허용
 - 공변: 구체적인 방향으로 타입 변환 허용 <? extends T>
 - 반공변: 추상적 방향으로 타입 변환을 허용 <? super T>

--- 
제네릭을 사용하는 이유

 - 형 변환이 필요 없고 타입 안정성 보장
 - 코드의 재사용성 증대.
